# User Onboarding

## Кому выдается

Инструкция выдается 3-4 участникам пилота после запуска OpenWebUI.

## Как войти

1. Открыть `https://gpt.alpha-soft.ru`.
2. Войти под выданным логином.
3. Сменить пароль, если это требуется политикой запуска.

## Как пользоваться

1. Создать новый чат.
2. Задать рабочий вопрос.
3. Уточнить ответ, если он неполный.
4. Сохранить полезные сценарии для feedback.

## Что нельзя отправлять

Не отправлять:

- пароли;
- токены;
- API-ключи;
- приватные SSH-ключи;
- закрытые договоры и персональные данные без отдельного разрешения.

## Куда обращаться

При проблемах обратиться к администратору пилота.
