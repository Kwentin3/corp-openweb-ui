# User Onboarding

## Кому выдается

Инструкция выдается 3-4 участникам пилота после запуска OpenWebUI.

## Как войти

1. Открыть `https://gpt.alpha-soft.ru`.
2. Войти под выданным логином.
3. Прочитать предупреждающий баннер.
4. Сменить пароль, если это требуется политикой запуска.

При входе или в интерфейсе пользователь увидит предупреждение "Тестовый корпоративный AI-чат". Его нужно соблюдать при каждом запросе.

## Как пользоваться

1. Создать новый чат.
2. Задать рабочий вопрос.
3. Уточнить ответ, если он неполный.
4. Сохранить полезные сценарии для feedback.

## Что нельзя отправлять

Не отправлять:

- пароли;
- токены;
- API-ключи;
- приватные SSH-ключи;
- закрытые договоры и персональные данные без отдельного разрешения.

Ответы модели нужно проверять. Нельзя использовать ответ как единственный источник для юридических, финансовых, кадровых или технически рискованных решений.

## Куда обращаться

При проблемах обратиться к администратору пилота.
