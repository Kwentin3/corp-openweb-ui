# Access Policy

## Роли

PRD-0 использует только две роли:

- administrator;
- user.

Сложные роли, РО/подчиненные и корпоративный RBAC не входят в PRD-0.

## Administrator

Администратор:

- управляет пользователями;
- настраивает модель;
- проверяет backup;
- выполняет smoke checks;
- собирает feedback.

## User

Пользователь:

- входит в OpenWebUI;
- задает рабочие вопросы;
- не загружает секреты;
- сообщает о проблемах администратору.

## Signup

Самостоятельная публичная регистрация должна быть отключена или ограничена. Если signup временно включается для создания пользователей, после создания пилотной группы его нужно отключить.
