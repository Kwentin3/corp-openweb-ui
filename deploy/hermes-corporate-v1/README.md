# Hermes Corporate v1 Deploy Placeholder

This directory is a placeholder for a future deployment scaffold.

It is not a production deployment, not a live runbook, and not an instruction to deploy. The current repository bootstrap task does not connect to a VPS and does not change any existing server.

## Intended Future Contents

- Concrete Compose contracts per node.
- Server-local environment layout.
- Per-node Authelia templates.
- Per-node Traefik naming conventions.
- Node registry.
- Runbooks for deploy, update, rollback, node removal and backup/restore.

## Safety

Do not place real secrets, real IP addresses, password hashes, OAuth secrets, API keys or private keys in this directory.
